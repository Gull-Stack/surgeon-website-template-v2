# Surgeon Website Template V2 — Tier S+

The ultimate plastic surgery website template. Luxury medical aesthetic design, config-driven, conversion-optimized, SEO-perfect.

## Quick Start

1. Edit `config.js` with your practice information
2. Replace placeholder images in `images/`
3. Deploy to Vercel, Netlify, or any static host

## Features

- **Config-Driven** — Change `config.js` to customize for any practice
- **Luxury Design** — Dark navy + gold palette, elegant typography, premium feel
- **Interactive B/A Slider** — Drag to compare before/after photos
- **Procedure Quiz** — "Find Your Procedure" interactive modal
- **Testimonial Carousel** — Auto-advancing, touch-friendly, with review source badges
- **Animated Counters** — Stats bar with counting animations
- **Full SEO** — Schema.org JSON-LD, Open Graph, meta tags, sitemaps
- **Mobile-First Responsive** — Looks great on every device
- **Accessibility** — WCAG 2.1 AA, skip links, keyboard navigation, ARIA labels
- **Performance** — Lazy loading, preconnect, efficient CSS

## Tech Stack

- Vanilla HTML5, CSS3, JavaScript (ES6+)
- No frameworks, no build tools, no dependencies
- Google Fonts: Inter, Playfair Display, Cormorant Garamond
- Font Awesome 6.x

## Pages

| Page | File | Description |
|------|------|-------------|
| Homepage | `index.html` | Full hero, stats, procedures, gallery, testimonials, quiz, financing |
| About | `about.html` | Doctor bio, education, awards, memberships |
| Contact | `contact.html` | Contact form, info sidebar, virtual consult |
| Gallery | `gallery.html` | Filterable B/A gallery with interactive sliders |
| Mommy Makeover | `services/mommy-makeover.html` | Service page template with FAQ, timeline, sidebar |

## Customization

All customization is done through `config.js`:
- Practice name, doctor info, contact details
- Services and procedures (enable/disable individually)
- Testimonials
- Color theme and typography
- Feature toggles (quiz, chat, virtual consult, etc.)
- SEO settings
